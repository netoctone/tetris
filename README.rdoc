== About

  Yet another tetris implementation. Now in pure ruby 1.9 using gosu.

== Installation

1. clone repo and cd into it

  git clone git://github.com/netoctone/tetris.git; cd tetris

2. install bundler (if you haven't yet)

  gem install bundler

3. install dependency gems

  bunlde install

4. enjoy the game (fix she-bang if needed)

  bin/tetris
